# 企业码数据分析自动化

企业码消费账单自动化分析报告生成系统

## 功能特性

- 支持 Excel/CSV 格式的对账单数据导入
- 自动生成多维度分析报告（省份/部门/商户/价格段/趋势）
- 输出标准化 H5 报告，支持浏览器直接打印为 PDF
- 图表自动渲染（基于 ECharts）
- 支持参数化配置（账期、数据源路径等）

## 目录结构

```
qiyema-report-gen/
├── index.html          # 分析报告 H5 页面
├── static/
│   └── images/         # 图表图片目录
├── data/               # 数据文件目录（对账单 Excel/CSV）
├── generate_report.py  # Python 数据分析脚本
└── README.md
```

## 使用方法

### 1. 安装依赖

```bash
pip install pandas openpyxl matplotlib
```

### 2. 生成报告

```bash
# 分析对账单数据
python3 generate_report.py <对账单文件路径> <输出目录>

# 示例
python3 generate_report.py ~/Downloads/对账单_2026年04月.xlsx ./
```

### 3. 查看报告

直接在浏览器中打开生成的 `企业码订单数据分析-YYYY-MM-DD.html` 文件

### 4. 导出 PDF

在浏览器中打开报告后，按 `Cmd/Ctrl + P` 打印，选择"保存为 PDF"，纸张方向选择"横向"。

## 报告结构

报告包含以下章节：

1. **KPI 总览** - 企业消费总额、订单笔数、客单价等核心指标
2. **消费结构总览** - 企业实付、非企业付、件单价
3. **消费价格段分布** - 各价格段订单笔数占比
4. **省份分析** - TOP15 省份消费排名 + 单均价排名
5. **部门分析** - TOP10 部门消费排名
6. **商户分析** - TOP20 商户 + 单均价排名
7. **补贴结构分析** - 企业补贴 vs 员工自付关系
8. **月度趋势** - 日均单价趋势、日消费额趋势
9. **综合洞察与建议** - 核心发现、风险预警、运营建议

## 数据要求

对账单 Excel 文件需包含以下字段：

| 字段名 | 说明 |
|--------|------|
| 支付（元） | 实际支付金额 |
| 退款（元） | 退款金额 |
| 订单原价 | 订单原价 |
| 非企业付金额 | 员工自付金额 |
| 部门 | 所属部门 |
| 消费商户名称 | 商户名称 |
| 消费所在省 | 消费者所在省 |
| 支付日期 | 订单日期 |

## 技术栈

- **前端**: HTML5 + CSS3 + 原生 JavaScript
- **图表**: ECharts 5.x
- **数据处理**: Python 3 + pandas + openpyxl + matplotlib
- **部署**: 静态文件服务器
